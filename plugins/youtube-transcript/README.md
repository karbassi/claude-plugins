# YouTube Transcript

Fetch and present transcripts from YouTube videos using yt-dlp.

## Requirements

- [yt-dlp](https://github.com/yt-dlp/yt-dlp) must be installed

## Usage

```
/transcript https://www.youtube.com/watch?v=VIDEO_ID
```

Or just provide a video URL and ask for the transcript.

## Skills

- **transcript** - Fetches YouTube video transcripts with automatic fallback strategies
