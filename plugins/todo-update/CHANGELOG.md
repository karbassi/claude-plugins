## [1.0.0-beta.2] - 2026-01-20

### Added

- Initial beta release
- Task tracking subagent using Haiku model for fast updates
- Structured TODO.md format with In Progress, Pending, Completed sections
- Checkbox syntax support (`- [ ]` and `- [x]`)
- Auto-approved file edits via `permissionMode: acceptEdits`
- Duplicate detection and stale entry cleanup
